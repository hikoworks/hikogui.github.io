Detailed Description

template<typename F, typename... Args>
requires invocable_task<F, std::stop_token, Args...>
class hi::v1::default_button_delegate< F, std::stop_token, Args... >

A default button delegate which handles the execution of a coroutine function returning a task<void> and taking a std::stop_token as the first argument.

A default button delegate which handles the execution of a function taking a std::stop_token as the first argument.

Template Parameters
FThe type of the function to execute.
ArgsThe types of the arguments to pass to the function.

The function will be called asynchronously using std::async() in a separate thread.
