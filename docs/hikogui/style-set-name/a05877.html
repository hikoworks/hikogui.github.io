std::vector<int8_t> v1::shaper_phase1_result::embedding_levels

The unicode-bid embedding level for each grapheme and each paragraph.

These are two lists appened together:

  • Embedding level for each grapheme, followed by
  • Embedding level for each paragraph.

The embedding levels can also be used to determine which brackets need to be mirrored (when the embedding level is odd).

unicode_word_break_vector v1::shaper_phase1_result::word_break_opportunities

A list of break opportunities where the edges of words are.

This list is used for selecting, or navigating words.

It is also used combined with the grapheme's attributes to make runs that can be shaped as a single entity by the font program.
