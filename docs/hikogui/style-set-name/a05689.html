std::expected< bookmark, std::error_code > v1::bookmark::resolve ( First && first,
Args &&... args ) const
inlinenodiscard

Resolve the bookmark to an actual path on the disk.

When resolving a bookmark with a given language and scale, a file without a language and scale needs to exist. This will help with searching quickly.

Files should have the following naming convention:

  • <path>/<name>(-<language>)(<scale>x).<ext>
Parameters
tagsWhen searching for a file, try to find a file matching the earliest of the languages-tags in the list.
scaleWhen searching for a file, try to find a file matching the given scale.
Returns
A bookmark that points to an actual file. or an error.
