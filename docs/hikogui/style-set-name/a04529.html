Detailed Description

template<typename Pair = std::pair<size_t, size_t>>
class v1::sequence< Pair >

A sequence of ranges.

Template Parameters
PairThe pair type of the range. Default is std::pair<size_t, size_t>. The first element is the first index of the range, and the second element is one beyond the last index of the range.
