translate3 v1::layout::shape::translate_z ( float offset = 0.0f) const
inlinenodiscardnoexcept

Get a translate object for the elevation of this shape.

This function will return a translate object that will translate the the z-coordinate to the elevation of the shape. This is used to draw geometric shapes at the correct elevation.

Parameters
offsetThe offset to add to the elevation.
Returns
A translate object that will translate the z-coordinate to the elevation.
