Detailed Description

A GUI widget that permits the user to make a binary choice.

A async is a button with three different states with different visual representation:

  • on: A check-mark is shown inside the box, and the async_widget::on_label is shown.
  • off: An empty box is shown, and the async_widget::off_label is shown.

Each time a user activates the async-button it toggles between the 'on' and 'off' states. If the async is in the 'other' state an activation will switch it to the 'off' state.

In the following example we create a async widget on the window which observes value. When the value is 1 the async is 'on', when the value is 2 the async is 'off'.
